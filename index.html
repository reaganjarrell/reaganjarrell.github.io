Unit 3

Throughout this unit we learned how to stylize our pages using CSS. CSS stands for Cascading Style Sheets. We learned multiple new skills such as how to float a paragraph and how to change the backround color. We used the Box Model to arrange our information into a fitted format. We also learned how to label our sections into groups, using class, and into individual sections, using id. This is all helpful when styling your webpage.

Unit 4

Throughout this unit we learned how to add images into our website in many different forms. We learned how to link images to different links, creating buttons. We also learned how to make a small image a thumbnail for a larger image. This incoorperated many different skills in photoshop and illustrator. Some were resizing, manipulating images, and saving under folder types. We also learned how to create and size banners to fitin our webpages.

Unit 5

Throughout this unit, we learned how to add buttons that allow a function. For example, we added a hide/show clock button. We learned how to reate the clock then how to give it a function. We also learned functions such as how to show a message using a button.

Unit 6

In this unit we focused on testing our websites. We tested video game students on our website. They filled out a form that containeed three tasks and two questions on our website. Then we took notes, reviewed, and fixed our website. This allowed us to user test and make sure our website was fully functioning with no errors.
