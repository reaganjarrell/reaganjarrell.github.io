Web Accessibility Checklist

Accessibility Review of the Your School Name Home Page

Throughout using accesibility evaluation tools, I've discovered many sorts of errors and places errors can be:

  • Errors

  • Alerts

  • Features

  • Structure Elements

  • HTML5 and ARIA

  • Contrast Errors

While testing the schools website, the WAVE mostly found errors in Structural Elements, HTML5, and ARIA. It shows me exactly where those errors are on the page, giving me a better chance to find them and fix them.

User characteristics Accessible design tips
Unable to see Code all Images with ATL text
Unable to percieve colors Avoid using color alone to convey information
Unable to use mouse Be sure all website features can be accessed using keyboard
Unable to hear Add captions to multimedia
Prone to having seizures Avoid content that flashes
Easily distractable Keep the design simple
