Evaluation of web graphics

Site Evaluated: Apple

Purpose of Graphics on this site

The graphics on the Apple home page are very useful to the overall look of the page. The site has a lot of white space, so the pictures offer a more interesting visual. The images switch every few seconds, offering a thurough advertisement on many Apple products. Each image links to it's own individual sell page, allowing the images to be not only visually appealing but useful. Apple effectively used graphics in their home page.

Guidelines for effective use of graphics

  1. Yes- Graphics should fit in with the purpose, organization, and style of the page.
  2. Yes- Avoid using graphics with large file sizes that add to the load time of the page.
  3. Yes- Graphics should help to guide the viewers’ focus to the important content on the page.
  4. Yes- Avoid repetitive use of overly bright or potentially “obnoxious” images.
  5. Yes- Avoid the use of graphics to convey textual content information.
  6. Yes- Provide textual equivalent alternatives for graphic content.
  7. Yes- When using text in graphics, make sure there is sufficient contrast between the text and the background color.
